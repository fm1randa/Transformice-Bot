-- Instructions --
	* Open keyparser.exe that is included in keyparser folder
	* Wait until Transformice window opens
	* Press Spacebar key on Transformice login interface.
	  It will open a popup message
	* Copy the entire popup message
	* Close Transformice window
	* Paste the message you copied into keyparser window and press Enter
	* Open bot.exe

++
	You can change bot username and password in config.txt file
	To add an admin into the admins list (in config.txt), you must use this format -> "Nick#tag": 0
	Example:
		"admins": {"Villaaam#0000": 0, "OtherName#0000": 0}
